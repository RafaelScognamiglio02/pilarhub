# Guia do Produto Web

Este projeto agora é um site web multiusuário.

## Primeiro acesso

1. Acesse `/login`.
2. Crie uma conta.
3. Abra `Configurações`.
4. Personalize nomes, cartões e projetos.
5. Comece a usar o Dashboard, Lançamentos, Cartões, Investimentos, Projetos e Metas.

## Modelo de uso

Cada pessoa cria uma conta própria. Os dados de uma conta não aparecem para outra.

## Deploy recomendado

- Vercel para o site.
- Neon ou Supabase para Postgres.
- Variável `DATABASE_URL` configurada no painel da hospedagem.
